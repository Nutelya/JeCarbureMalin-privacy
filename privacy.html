Politique de confidentialité

Application : Je Carbure Malin
Contact : contact@jecarburemalin.fr
Dernière mise à jour : avril 2025

1. Données collectées

Je Carbure Malin ne collecte aucune donnée personnelle identifiable. Aucun compte utilisateur n'est requis.

2. Localisation

L'application utilise votre position GPS uniquement pour afficher les stations proches. Cette donnée n'est jamais transmise à nos serveurs ni stockée.

3. Données stockées localement

Vos préférences (carburant, rayon, filtres, marques masquées) sont enregistrées localement sur votre appareil via AsyncStorage. Ces données ne quittent jamais votre appareil.

4. Authentification anonyme

L'application utilise Firebase Authentication (Google) pour une connexion anonyme afin de sécuriser les échanges avec notre API. Aucune information personnelle n'est collectée lors de cette authentification.

5. Publicité

L'application affiche des publicités via Google AdMob. Google peut utiliser des identifiants publicitaires (GAID) pour personnaliser les annonces. Vous pouvez limiter ce suivi dans Paramètres → Confidentialité → Publicité de votre appareil.

6. Services tiers

  • data.economie.gouv.fr — Prix des carburants (données publiques)
  • OpenStreetMap / Overpass API / Nominatim — Cartographie et identification des marques
  • Google Places API — Identification des marques (côté serveur uniquement)
  • OSRM — Calcul d'itinéraires (auto-hébergé)
  • Firebase Authentication (Google) — Authentification anonyme
  • Google AdMob — Publicité

7. Vos droits (RGPD)

Conformément au RGPD, vous disposez d'un droit d'accès, de rectification et de suppression des données vous concernant. Contactez-nous à contact@jecarburemalin.fr.
